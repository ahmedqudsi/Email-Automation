# Email Automation

# Key Features

-> Sends emails one by one to multiple recipients<br>
-> Supports file attachments (resume, documents, etc.)<br>
-> Uses TLS encryption for secure email transmission<br>
-> Includes a time delay between emails to avoid spam flags<br>
-> Easily customizable subject, body, and recipient list<br>

# Technologies Used

->Python<br>
->smtplib<br>
->email.mime modules<br>
->SMTP (Gmail server)<br>


# Use Case:
->Job applications<br>
->Internship outreach<br>
->Professional cold emailing with attachments<br>

# How It Works
->Configure sender email and app password<br>
->Add recipient email addresses<br>
->Set email subject, body, and attachment path<br>
->Run the script to send emails automatically<br>

# Note
For Gmail, use an App Password instead of your main account password.
