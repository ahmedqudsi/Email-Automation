# Email Automation

# Key Features

-> Sends emails one by one to multiple recipients
-> Supports file attachments (resume, documents, etc.)
-> Uses TLS encryption for secure email transmission
-> Includes a time delay between emails to avoid spam flags
-> Easily customizable subject, body, and recipient list

# Technologies Used

->Python
->smtplib
->email.mime modules
->SMTP (Gmail server)


# Use Case:
->Job applications
->Internship outreach
->Professional cold emailing with attachments

# How It Works
->Configure sender email and app password
->Add recipient email addresses
->Set email subject, body, and attachment path
->Run the script to send emails automatically

# Note
For Gmail, use an App Password instead of your main account password.
